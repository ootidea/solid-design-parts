# solidjs-skel

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

## Quick start

Install it:

```bash
npm i solidjs-skel
# or
yarn add solidjs-skel
# or
pnpm add solidjs-skel
```

Use it:

```tsx
import solidjs-skel from 'solidjs-skel'
```
