# solidjs-skel

## Quick start

Install it:

```bash
npm i solidjs-skel
# or
yarn add solidjs-skel
# or
pnpm add solidjs-skel
```

Use it:

```tsx
import solidjs-skel from 'solidjs-skel'
```
